# Installation

## Emacs 24

Download `twlight-theme.el` to the directory `~/.emacs.d/themes/`. Add this to your
`.emacs`:

`(add-to-list 'custom-theme-load-path "~/.emacs.d/themes/")`

Now you can load the theme with the interactive function `load-theme`.
